# Getting Started

At the terminal, run:

```bash
git clone git@github.com:dev-br/logo.git
cd logo
yarn
yarn start
```

Output similar:

```bash
yarn run v1.22.4
$ http-server ./ -p 8085
Starting up http-server, serving ./
Available on:
  http://127.0.0.1:8085
  http://192.168.9.175:8085
  http://172.18.0.1:8085
Hit CTRL-C to stop the server
```
